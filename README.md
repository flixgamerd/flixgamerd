## 🚧 Projeto em Destaque — usina-monitor

> Sistema de monitoramento industrial para dessalinização de água via ESP32.

Um sistema embarcado completo com pipeline de dados do sensor à dashboard:

```
ESP32 (C++) → MQTT → Node.js/TypeScript → PostgreSQL → Qt Dashboard
```

- 📡 Leitura de **pH** e **salinidade** em tempo real
- 🔁 Arquitectura **Layered + Event-Driven** no backend
- 🤖 Integração com **IA** (Claude Sonnet) para análise de anomalias
- 📊 Dashboard com modo de dados simulados

🔗 [Ver repositório](https://github.com/flixgamerd/usina-monitor)
